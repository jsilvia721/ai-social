# Strategy Digest UI — Brainstorm

**Date:** 2026-03-08
**Status:** Complete
**Next step:** `/ce:plan`

## What We're Building

A read-only `/dashboard/insights` page that surfaces the weekly AI performance digests generated by the M3 strategy optimizer. The primary user is the non-technical business partner who wants to understand what's working without digging into raw metrics.

### Core Experience

- **Latest-focused layout** — most recent digest shown prominently with a week-picker to navigate older ones
- **Four content sections per digest:**
  1. **Summary** — plain-language overview (already in `StrategyDigest.summary`)
  2. **Top Performers** — best posts with engagement scores (from `patterns.topPerformers`)
  3. **Key Insights** — bullet-point observations (from `patterns.insights`)
  4. **Strategy Adjustments** — what the optimizer changed (from `changes.formatMix`, `changes.cadence`, `changes.topicInsights`)
- **Read-only** — the optimizer auto-applies changes within guardrails, no user action needed

## Why This Approach

- The optimizer already runs autonomously and applies changes. Adding an approval gate would add friction and defeat the "80% autonomous" goal.
- Latest-focused is better than a feed because the partner cares about "what happened this week" — historical comparison is secondary.
- Existing `GET /api/businesses/[id]/digests` endpoint returns the last 4 digests, so the data layer is already done.

## Key Decisions

1. **Read-only** — no approve/reject workflow. The optimizer auto-applies within guardrails.
2. **Latest-focused with week picker** — not a scrollable feed. Show one digest at a time.
3. **Server component** — digests are read-only, no interactivity beyond week selection. Week picker can be a client component island.
4. **Reuse existing API** — `GET /api/businesses/[id]/digests` already returns what we need.
5. **Add sidebar nav entry** — "Insights" link with a distinguishing icon (e.g., `Lightbulb` from lucide-react).
6. **Empty state** — friendly message when no digests exist yet ("Your first weekly insight will appear after the optimizer runs").
